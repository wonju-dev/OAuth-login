# OAuth-login

Passport.js 없이 OAuth를 통한 로그인을 구현해보자

[X]TypeScript 환경 설정 하기

- npm init -y
- yarn add typescript
- tsc --init
  - tsconfig 파일 생성, 설정

[X] nodemon 설치

- tsc는 트랜스파일링만 수행
- ts-node를 통해 트랜스파일링 + 실행
  - yarn add ts-node
- yarn add nodemon
- 스크립트 작성
  - nodemon --exec ts-node 실행할파일.ts
